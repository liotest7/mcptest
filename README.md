# Installation Steps

To install the 'add_tool' MCP Server, run the following command
```json
{
    "mcpServers": {
        "server": {
            "command": "uvx",
            "args": [
                "--from",
                "git+https://github.com/liotest7/mcptest.git",
                "mcp-server"
            ]
        }
    }
}
```
